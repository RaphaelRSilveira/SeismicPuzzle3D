# Seismic Puzzle 3D

O **Seismic Puzzle 3D** (também conhecido como *SeismicPuzzle* ou *Seismic Puzzle*) é uma ferramenta web interativa que lê arquivos de horizontes sísmicos e os transforma em modelos 3D (quebra-cabeças geológicos) prontos para impressão em impressoras 3D. 

Ideal para geólogos, geofísicos, educadores e entusiastas que desejam materializar dados sísmicos estruturais em objetos físicos e interativos.

## Funcionalidades
- **Importação de Horizontes:** Lê arquivos de texto contendo dados de horizontes sísmicos.
- **Visualização 3D:** Permite a exploração interativa das camadas geológicas e falhas no navegador.
- **Exportação para Impressão 3D:** Gera arquivos STL e 3MF otimizados e prontos para serem fatiados e impressos.
- **Personalização:** Ajuste de escala, espessura da base, cores, e adição de legendas e textos em relevo.
- **Internacionalização:** Suporte para múltiplos idiomas (Inglês, Português, Espanhol, Francês e Italiano).
